## About

SIMPRA - Sistem Keuangan RSUD Soetomo

## Setup

Make sure to install the dependencies:

```bash
composer install
```

Update packages

```bash
composer update
```

Update Node Module

```bash
npm install
```

## Development Server

Start the development server on http://localhost:8000

```bash
php artisan serve
```
